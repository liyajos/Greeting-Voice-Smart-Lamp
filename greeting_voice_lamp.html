Greeting Voice Smart Lamp

Hold a button, say something, and an RGB LED answers — in color and in kind. A five-step build from "can it hear me" to "it says good morning back."

How it's built ↓
Overview

Three moves, every time

Every interaction runs the same loop: listen while the button is held, send the clip to Google's speech API, then react — either with a color, or with a greeting and a little light show.

01 · Listen

Push-to-talk capture

Holding the button on GPIO 21 opens the mic. recognizer.listen() records until release or a 5 second timeout, with ambient noise calibrated once at startup.

02 · Understand

Keyword parsing

The transcript is lower-cased and scanned for keywords — a color name, "on"/"off", "rainbow", or a greeting phrase like "good morning." First match wins.

03 · Glow

LED as feedback

The RGB LED on pins 17/27/22 doubles as a status light: dim blue while listening, dim yellow while processing, then a flash of green, red, or orange for the result.

Build log

Five steps, in order

Each file in the repo is a working checkpoint — the project actually ran at every stage below.

1

Just listenstep2.py

No LED, no button yet. Calibrate for ambient noise, record 5 seconds, send it to recognize_google(), print whatever came back — or the specific reason it failed.

2

Voice becomes colorstep3.py

The RGB LED joins in. A dictionary of 18 named colors plus "on," "off," and "rainbow" get matched against the transcript and pushed straight to led.color, in a free-running loop.

3

Push-to-talkstep4.py

The always-on mic gets replaced with a button. wait_for_press() gates listening, and the LED dims blue while held so you know it's actually recording.

4

A light for every outcomestep5.py

Distinct states for idle, listening, processing, success, misheard, and unmatched, plus a last_color memory so a flash of feedback doesn't erase the color you just asked for.

5

It talks backextension_greeting.py

Greeting phrases — "hello," "good morning," "thank you" — get their own printed replies and short signature blinks, on top of the full color and state-feedback system from step 5.

def parse(text):
    """Look at the text and decide what command was given"""
    t = text.lower()  # Convert to lowercase so "Red" and "red" both work
    
    if "off" in t:
        return ("off", None)
    if "on" in t:
        return ("on", (1, 0.7, 0.3))  # Warm white when turned on
    for name, rgb in COLORS.items():

    if "rainbow" in t or "party" in t:

    return (None, None)  # No matching command found

from gpiozero import RGBLED, Button
from time import sleep

button = Button(21)

mic = sr.Microphone()

COLORS = {
    "red": (1, 0, 0),
    "green": (0, 1, 0),
    "blue": (0, 0, 1),
    "yellow": (1, 1, 0),
    "purple": (0.5, 0, 1),
    "white": (1, 1, 1),
    "orange": (1, 0.5, 0),
    "pink": (1, 0.4, 0.6),
}

last_color = None

# -------------------------------
# Parse Voice Commands
# -------------------------------
def parse(text):
    t = text.lower()

    if "off" in t:
        return ("off", None)

    if "on" in t:
        return ("on", (1, 0.7, 0.3))

    if "hello" in t or "hi" in t:
        return ("hello", None)

    if "good morning" in t:
        return ("morning", None)

    if "good afternoon" in t:
        return ("afternoon", None)

    if "good evening" in t:
        return ("evening", None)

    if "good night" in t:
        return ("night", None)

    if "thank you" in t:
        return ("thanks", None)

    for name, rgb in COLORS.items():
        if name in t:
            return ("color", rgb)

    if "rainbow" in t or "party" in t:
        return ("rainbow", None)

    return (None, None)


# -------------------------------
# Execute Commands
# -------------------------------
def execute(cmd, arg):
    global last_color

    if cmd == "off":
        last_color = None
        led.off()

    elif cmd in ("on", "color"):
        last_color = arg
        led.color = arg

    elif cmd == "rainbow":
        for c in COLORS.values():
            led.color = c
            sleep(0.3)
        led.off()

    elif cmd == "hello":
        print("Hello! Nice to meet you!")
        for _ in range(2):
            led.color = (0, 1, 0)
            sleep(0.3)
            led.off()
            sleep(0.3)

    elif cmd == "morning":
        print("Good Morning!")
        led.color = (1, 1, 0)
        sleep(2)
        led.off()

    elif cmd == "afternoon":
        print("Good Afternoon!")
        led.color = (1, 0.5, 0)
        sleep(2)
        led.off()

    elif cmd == "evening":
        print("Good Evening!")
        led.color = (1, 0, 1)
        sleep(2)
        led.off()

    elif cmd == "night":
        print("Good Night!")
        led.color = (0, 0, 1)
        sleep(2)
        led.off()

    elif cmd == "thanks":
        print("You're Welcome!")
        for _ in range(3):
            led.color = (0, 1, 0)
            sleep(0.2)
            led.off()
            sleep(0.2)
